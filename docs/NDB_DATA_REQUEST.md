### B. 健診検査値 5項目 — Bridge直結の最重要セット (優先度: **最高**)

| 項目 | 単位 | 用途 (Bridge) | リスク閾値案 |
|---|---|---|---|
| **HbA1c** (NGSP) | % | 糖尿病 | ≥6.5% (糖尿病型), ≥7.0% |
| **BMI** | kg/m² | 糖尿病・代謝・循環器 | ≥25 (肥満), ≥30 |
| **収縮期血圧** | mmHg | 循環器・脳血管 | ≥140 (高血圧), ≥130 |
| **LDLコレステロール** | mg/dL | 循環器 | ≥140 (脂質異常症), ≥160 |
| **尿蛋白** | 定性5段階 | 腎疾患 | 1+以上 |

**取得元**: 厚労省 [第10回NDBオープンデータ](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/0000177182.html) の「特定健康診査結果」  
**理想形式**: 都道府県 × 男女 × 年齢階級5歳刻み (40-44, 45-49, ..., 70-74) **× 階級値分布** (例: BMI<18.5 / 18.5-25 / 25-30 / 30+ の人数)

**フォールバック**: 階級分布が無ければ平均値版でも可 (`NDB_特定健診_平均値_都道府県別.xlsx` の拡張版に上記5項目が含まれていれば即利用)

## 妥協案 (B 階級分布が入手不可な場合)

平均値のみで実装する場合のリスク提示は限定的:
- **平均値の県差** = 「閾値超過率の県差」とは言えないが、傾向は反映
- 例: 「県平均 SBP=135mmHg」→「高血圧傾向あり」程度の表現
- 閾値ベースの厳密な「リスク該当者比率」は無理

→ 階級分布が**理想**だが、**平均値だけでも進める価値はある** (現状3項目→8項目に拡張)
