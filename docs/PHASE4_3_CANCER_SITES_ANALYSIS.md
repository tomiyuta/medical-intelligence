# Phase 4-3: 5 大がん部位別 dispersion 分析

**作成日**: 2026-04-30
**branch**: feature/phase4-3-cancer-sites
**目的**: 「ガン全部位の合算では県差が小さい」(`docs/ANALYSIS_MORTALITY_DISPERSION.md`) の続編。**部位別では本当に県差が大きいのか** を実データで検証。
**結論サマリ**: ✅ **仮説完全実証**。部位別で見ると CV が 2-2.5 倍に拡大。合算で打ち消されていた **真の地域差が顕在化**。特に女性肝がんは max-min 比 **12.56 倍**。

---

## 0. Executive Summary

| 軸 | 全部位 | 部位別 (max) | 拡大倍率 |
|---|---|---|---|
| CV (男女計) | 8.77% | **21.52%** (前立腺) | **2.45 倍** |
| max-min 比 (男女計) | 1.56 | **3.71** (前立腺) | **2.38 倍** |
| 女性 CV | 9.11% | **32.64%** (肝) | **3.58 倍** |
| 女性 max-min 比 | 1.50 | **12.56** (肝) | **8.37 倍** |

**重要発見**: 5 大がんは **部位別に異なる地域パターン** を持ち、合算で打ち消されている:
- **胃** → 秋田 (東北・塩分)
- **肝** → 青森 (HBV/HCV)
- **大腸** → 青森
- **肺** → 北海道 (喫煙)
- **乳房** → 北海道
- **前立腺** → 香川 (高位)、奈良 (低位)

## 2. 検証結果 (2024 年、男女計)

### 2.1 dispersion ranking

| 順位 | 部位 | CV | max-min 比 | 全国平均 | 高位県 | 低位県 |
|---|---|---|---|---|---|---|
| 1 | **前立腺 (男)** | **21.52%** | **3.71** | 2.1 | 香川 (3.3) | 奈良 (0.9) |
| 2 | **肝** | **18.54%** | **2.60** | 3.2 | 青森 (4.5) | 福井 (1.7) |
| 3 | **乳房 (女)** | **16.47%** | **2.50** | 9.6 | 北海道 (12.8) | 徳島 (5.1) |
| 4 | **大腸** | **15.52%** | **2.28** | 9.8 | 青森 (15.3) | 滋賀 (6.7) |
| 5 | **胃** | **14.78%** | **1.89** | 5.6 | 秋田 (7.6) | 熊本 (4.0) |
| 6 | **肺** | **14.13%** | **1.99** | 11.0 | 北海道 (15.7) | 山梨 (7.9) |
| **(参考)** | **全部位** | **8.77%** | **1.56** | 65.3 | 青森 (84.2) | 長野 (53.8) |

→ **5 大がんすべて、全部位より CV が 1.6-2.5 倍大きい**。

### 2.2 性別による違い (女性肝がんの異常値)

| 部位 | 男 CV | 女 CV | 女 max-min 比 |
|---|---|---|---|
| 胃 | 16.05% | **20.22%** | 2.12 |
| 大腸 | 18.34% | 15.57% | 2.16 |
| 肝 | 19.53% | **32.64%** | **12.56** ★ |
| 肺 | 14.74% | **21.86%** | 2.86 |

**女性肝がん**: CV 32.64% / max-min 比 **12.56 倍** = 青森 (1.7) vs 福井 (0.13)。
- mean が 1.3/10万人と非常に小さく、絶対数が小さいので CV/比が大きく出る統計 artifact 候補
- ただし HCV/HBV 感染率の地域差 (西日本 + 一部東北) を反映している可能性あり

### 2.3 地域パターン (高位県の食い違い)

5 大がんの「最も県差が大きい部位」が **県によって違う** ことが、合算で県差が消える理由:

| 県 | 高位部位 | 仮説 |
|---|---|---|
| 青森 | **肝・大腸 (1位)** | HBV/HCV、食生活 |
| 秋田 | **胃 (1位)** | 塩分文化、東北 |
| 北海道 | **肺・乳房 (1位)** | 喫煙率、出産・肥満 |
| 香川 | **前立腺 (1位)** | (要調査) |
| 和歌山 | **男肺 (1位)** | 喫煙率 |
| 鳥取 | **男肝 (1位)** | (要調査) |
| 滋賀 | 大腸最低 | 健診率高、健康行動 |
| 長野 | **全部位最低** | 食生活、長寿県 |

→ 異なる地域差軸が同時並行で存在 → 合算で互いに打ち消す。

## 3. 仮説完全実証 (前 docs `ANALYSIS_MORTALITY_DISPERSION.md` の反論 4 への応答)

前 docs (`ANALYSIS_MORTALITY_DISPERSION.md`) で reviewer 想定 反論 4 として:

> **反論 4**: 「ガンの部位別 (5 大がん) なら県差大では?」
> → **正しい**。ガン全体の CV が小さいのは部位平均の打ち消し効果 (推測)。

**この仮説が data で完全実証された**:
- 全部位 CV 8.77% → 部位別では 14-22% (1.6-2.5 倍)
- 女性肝がん CV 32.64% は 全部位 CV の **3.7 倍**
- 部位別の「地域差軸」は県によって違う

## 4. MedIntel への示唆

### 4.1 ガンの県差は「ある」が、表示単位による

| 表示単位 | dispersion | 解釈 |
|---|---|---|
| 全がん合算 | CV 8.77% (低い) | 県差は小さく見える |
| **5 大がん部位別** | CV 14-22% (大きい) | **県差は他疾患並みかそれ以上** |

→ **ユーザーの体感「ガンには県差がある」は部位別レベルでは正しい**。ただし、それを示すには UI で **部位別表示が必要**。

### 4.2 既存 mortality_outcome_2020.json との関係

|  | 既存 (粗+年齢調整、1985 model) | 新規 cancer_sites_mortality_2024.json |
|---|---|---|
| 期間 | 2020 単年 | 2024 (+ 2020-2024 5 年平均) |
| 部位 | 6 死因 (悪性新生物 = 全部位のみ) | 5 大がん部位別 + 全部位 |
| 標準化 | 全年齢年齢調整 | **75 歳未満** 年齢調整 |
| 互換性 | △ 全部位の年齢範囲が違う |

⚠ **重要 caveat**: 既存 data は全年齢、新規 data は 75 歳未満限定。直接比較は不可、独立 metric として扱う。
