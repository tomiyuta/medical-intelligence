# 主要死因の県差 dispersion 分析 — 「体感」と data の乖離

**作成日**: 2026-04-30
**branch**: feature/mortality-dispersion-analysis
**目的**: ユーザー観察「各県の10万人あたりのガン死亡者数は差が大きい、それ以外はあまり差がない」を MECE に検証
**結論サマリ**: ⚠ **ユーザーの体感は data 上 完全に逆**。47 県の dispersion を CV (変動係数) で比較すると、**悪性新生物 (ガン) は 6 死因中 最も県差が小さい (1 位)**。糖尿病・肺炎・脳血管疾患の方が 2-3 倍 dispersion が大きい。本 docs ではこの「逆認知」を MECE に分解する。

---

## 0. Executive Summary

| 軸 | 結論 |
|---|---|
| ユーザー観察 | 「ガンだけ県差大、他は小さい」 |
| **data 上の実態** | **ガンは 6 死因中 県差が最小** |
| 真に県差が大きい死因 | 糖尿病 / 肺炎 / 脳血管疾患 (CV はガンの 2-3 倍) |
| 主因 | base rate effect + 注目バイアス + UI color scale 設計 |
| MedIntel への示唆 | 死因比較 UI に **CV または z-score 標準化** の併記が必要 |

## 1. 検証: 47 県 dispersion 実測 (`mortality_outcome_2020.json`)

### 1.1 出典と data 構造

`data/static/mortality_outcome_2020.json`:
- source: 令和5年度人口動態統計特殊報告 令和2年都道府県別年齢調整死亡率の概況
- year: 2020
- unit: 人口10万対
- causes (6): 悪性新生物 / 糖尿病 / 心疾患 / 脳血管疾患 / 肺炎 / 腎不全
- mode: crude (粗死亡率) / age_adjusted (1985 モデル人口で年齢調整)
- prefectures: 47 県、男女別

### 1.2 CV (変動係数) ranking — 年齢調整 男性

CV (Coefficient of Variation) = standard deviation / mean × 100

**CV が小さい = 県差が小さい**:

| 順位 | 死因 | CV | max-min 比 | 全国平均 | max県 / min県 |
|---|---|---|---|---|---|
| **1** | **悪性新生物** | **6.37%** | **1.397** | 392.4 | 青森 / 長野 |
| 2 | 心疾患 | 10.29% | 1.631 | 189.7 | 愛媛 / 福岡 |
| 3 | 腎不全 | 12.72% | 1.995 | 27.8 | 青森 / 長野 |
| 4 | 脳血管疾患 | 14.49% | 1.932 | 97.4 | 岩手 / 滋賀 |
| 5 | 肺炎 | 17.32% | 2.045 | 89.7 | 青森 / 島根 |
| 6 | 糖尿病 | 19.37% | 2.208 | 14.5 | 香川 / 神奈川 |

→ **ガンの CV はワースト 6 位 (糖尿病) の約 1/3、最大-最小比は 1.40 倍に対し糖尿病は 2.21 倍**。

### 1.3 全パターンで同じ結論

| | ガン CV | ガン順位 | ワースト疾患 |
|---|---|---|---|
| 粗死亡率 男 | 12.33% | **1 位** | 肺炎 (22.58%) |
| 粗死亡率 女 | 12.15% | **1 位** | 肺炎 (29.52%) |
| 年齢調整 男 | 6.37% | **1 位** | 糖尿病 (19.37%) |
| 年齢調整 女 | 5.94% | **1 位** | 肺炎 (22.69%) |

→ **4 パターンすべてで「ガンが最小県差」が再現**。

### 1.4 真に県差が大きい死因の例 (年齢調整 男性)

| 死因 | max-min 比 | 高位県 | 低位県 |

## 2. なぜユーザーは「ガンだけ差が大きい」と感じたのか (MECE 分析)

### 2.1 軸 A: 視覚・UI の問題 (見かけの差を生む要因)

#### A1. **Base rate effect (絶対値の大きさ)** — 最有力候補

ガンの全国平均 (392) は糖尿病 (14.5) の **27 倍**。同じ CV (10%) でも:
- ガン CV 10% → 全国 ±39 の範囲 (絶対差 80)
- 糖尿病 CV 10% → 全国 ±1.5 の範囲 (絶対差 3)

→ map 上の数値・テキスト表示で **絶対差を見ると** ガンの差が大きく見えるが、これは CV ではなく base rate の違い。

#### A2. **Color scale の設計**

MedIntel の死因 map (推測) は、各疾患独立に min-max で 5 階の color scale。
- ガン: [338, 473] → 階あたり +27
- 糖尿病: [9.6, 21.2] → 階あたり +2.3

**色階で見れば全疾患で「赤い県/青い県」の比率は同等のはず**。ユーザーが「ガンだけ差が大きい」と感じたのは:
- 数値表示 (tooltip 等) で絶対差を読んだ可能性 (= base rate effect)
- または map 並列表示時に **ガンの色面積が広く見えた** (CV 差ではなく描画 area の問題)

→ **MedIntel の現在の色階設計を要確認**。

#### A3. **粗死亡率 vs 年齢調整死亡率の差**

ガンは年齢調整で CV が **半減** (12.33% → 6.37%)。
他疾患は半減ほどではない (心疾患 15.83% → 10.29% = 65% 残る)。

→ もし MedIntel が **粗死亡率を表示** している場合、ガンの「県差」は **高齢化率の県差を反映している** だけで、真の地域差ではない可能性。
→ これでも「ガンが最大」ではないが、CV 差が縮まる。

#### A4. **Anchoring (最初に目に入った疾患)**

UI で死因が「悪性新生物 → 心疾患 → ...」順に並ぶ場合、最初の悪性新生物の印象が強く残る。anchor 効果でユーザーは「ガンの差が際立つ」と感じる。

### 2.2 軸 B: 認知・期待バイアス

#### B1. **ガンへの社会的注目**

- 死因第 1 位 (全死亡の 27.6%、人口10万対 307)
- メディア・健康政策の中心トピック
- 「がん検診」「がん 5 年生存率」「がん登録」など制度化された情報

→ ユーザーは「ガンは重要 = 県差もあるはず」と無意識に期待 (confirmation bias 候補)。

#### B2. **既知の地域差ストーリーの引きずり**

- 「秋田・東北は胃がん多い」(塩分文化)
- 「肝がんは西日本」(C 型肝炎ウイルス)
- 「肺がんは喫煙率に比例」

これら部位別ナラティブから「ガン全体も県差大」と一般化。実際は 5 部位の合算で **平均化されて差が縮小** する。

#### B3. **他疾患の地域差ストーリー欠如**

- 糖尿病: CV 19% で県差最大級だが、ユーザーが認識する「県差」ナラティブは弱い
- 肺炎: CV 17%、誤嚥性肺炎の集計差 (Phase 2E-3 で議論済) があるが報道少ない
- 脳血管疾患: 「東北は脳卒中多い」のナラティブはあるが、近年の救急体制で差が縮小したと認識されやすい

→ ユーザーは「ガンに県差がある」を覚えているが「他の疾患にも県差がある」を覚えていない。

### 2.3 軸 C: 統計的 artifact (指標の選び方による artifact)

#### C1. **絶対差 vs 相対差 (CV)**

| 比較 | ガン | 糖尿病 |
|---|---|---|
| 絶対差 (max - min) | 134.2 | 11.6 |
| 相対差 (CV) | 6.37% | 19.37% |

絶対差で見るとガンが大、CV で見ると糖尿病が大。**どちらの指標で評価するかで結論が逆**。
ユーザーは無意識に絶対差で判断した可能性が高い (人口10万対の数値表示は絶対差を強調)。

#### C2. **年齢調整の効果は疾患で違う**

|  | ガン | 心疾患 | 脳血管 | 糖尿病 |
|---|---|---|---|---|
| 粗死亡率 CV (男) | 12.33% | 15.83% | 20.39% | 22.01% |
| 年齢調整 CV (男) | **6.37%** | 10.29% | 14.49% | 19.37% |
| 縮小率 | **48%** | 35% | 29% | 12% |

**ガンは年齢構成の影響が最も大きい疾患** (年齢調整で CV が半減)。
逆に **糖尿病は年齢調整しても県差が残る** (生活習慣・食文化の地域要因が強い)。

→ 年齢調整死亡率を見るかどうかで「どの疾患が県差大」の答えが変わる。

### 2.4 軸 D: 見落としやすい真の地域差軸 (ユーザーが認識すべき軸)

ユーザーが「他は差がない」と感じた疾患のうち、実際は大きな差がある:

#### D1. 糖尿病 (CV 19.37% 男)
- 香川 21.2 vs 神奈川 9.6 = 2.21 倍
- 解釈仮説: 食塩・うどん等の食文化、運動量、肥満率
- 注目度: 低い (急性死亡が少ないため)
- → MedIntel の Layer 1 ROOT CAUSE で BMI/HbA1c の県差を扱っているが、糖尿病死亡率の県差は表示されているか? (UI 確認候補)

#### D2. 肺炎 (CV 17.32% 男)
- 青森 118.4 vs 島根 57.9 = 2.05 倍
- 解釈仮説: 高齢化、気候、誤嚥性肺炎の集計差 (ICD-10 改訂影響)
- 注目度: 中 (高齢化議論で扱われる)
- caveat: Phase 2E-3 で「誤嚥性肺炎の死亡診断書記載差」を議論済

#### D3. 脳血管疾患 (CV 14.49% 男)
- 岩手 147.2 vs 滋賀 76.2 = 1.93 倍
- 解釈仮説: 食塩、寒冷気候、Phase 4-1 P5 (Aging-Outcome Burden) で扱った
- 注目度: 高い (P5 代表例)
- → MedIntel の Pattern 5 で既に扱っているが、ユーザーは「他は差がない」と感じている = UI で目立っていない

#### D4. 腎不全 (CV 12.72% 男)
- 青森 37.9 vs 長野 19.0 = 1.99 倍
- 解釈仮説: 透析患者数の地域差、糖尿病性腎症と相関
- 注目度: 低い

## 3. MedIntel UI への示唆

### 3.1 現状の表示の課題 (推測)

ユーザーが「ガンだけ差が大きい」と感じた事実は、UI が **「県差の大きさ」を正しく伝えられていない** ことを示唆。可能な原因:
- (a) 絶対値の人口10万対を表示 → base rate effect で大絶対値疾患 (ガン) が際立つ
- (b) 各疾患独立 color scale → 色階の差は出るが、ユーザーは色面積で判断する
- (c) 数値 + 色の併用 → 数値 (絶対差) と色 (相対差) の伝達が混在

### 3.2 改善候補 (実装は別フェーズ判断)

| 案 | 内容 | 効果 | リスク |
|---|---|---|---|
| **R1** | CV / max-min 比 / IQR を **「県差度」 KPI** として併記 | 体感バイアスを is data 上 修正 | 実装小、教育 UI 必要 |
| R2 | z-score 標準化 map (全疾患で同じ ±2σ scale) | 全疾患を同じ尺度で比較 | UI 複雑化 |
| R3 | 粗死亡率 / 年齢調整死亡率を toggle で切替 | 年齢構造の影響を可視化 | 既存 UI 変更必要 |
| R4 | 「県差ランキング (CV 順)」card を追加 | 一目で「どの疾患が地域差大」が分かる | 別 view 化が必要 |

### 3.3 Phase 4-1 guardrail との整合

- ✅ ranking 表示 (R4) は **疾患の県差大小** であり、**県の優劣** ではない (guardrail 抵触せず)
- ⚠ ただし「県差大の疾患 = 重要、県差小の疾患 = 不重要」という誤読を生む可能性
- 推奨 caveat: 「県差の大小は **介入余地の指標** であり、疾患の重要度ではない」

## 4. devil's advocate (本分析への反論)

### 反論 1: 「CV だけが正しい指標とは限らない」

→ 正しい。絶対死亡数で見ると依然ガンが圧倒的多数 (37.8 万人 vs 糖尿病 1.4 万人)。
公衆衛生介入の優先度は **絶対死亡数 × 削減可能性** で決まる。CV はあくまで「県間の介入余地の不均一性」の指標。

### 反論 2: 「年齢調整で見る合理性は?」

→ 県は年齢構成が大きく異なる (秋田 75歳以上 18% vs 沖縄 12%)。粗死亡率は年齢構成の差を反映するため、**真の地域要因 (食生活・医療・環境) を比較するには年齢調整が必須**。
ただし「実際に各県で死んでいる人数の負荷」を見るには粗死亡率が直感的。両方の併記が望ましい。

### 反論 3: 「6 死因に老衰・自殺が含まれていない」

→ 正しい。`mortality_outcome_2020.json` には老衰・自殺が含まれていない。
- 老衰 全国 107.5 (人口10万対) は 2020年 第 3 位、地域差大 (秋田 vs 沖縄 で 2-3 倍と推定)
- 自殺 全国 16.7、地域差中 (秋田 vs 神奈川)
- → これらを含めると「ガンが最小県差」結論はさらに強化される (老衰・自殺の方が CV 大)

### 反論 4: 「ガンの部位別 (5 大がん) なら県差大では?」

→ 正しい。**ガン全体の CV が小さいのは部位平均の打ち消し効果**:
- 胃がん: 東北高位、CV 大
- 肝がん: 西日本高位、CV 大
- 肺がん: 喫煙率と相関、CV 中
→ 部位別 ETL があれば各 5 部位でガンも県差が大きいことが判明する。
→ Phase 4-3 の今後の sub-topic 候補。

## 8. まとめ

**ユーザーの体感「ガンだけ県差大」は、47 県 data で完全に否定された**。実際は:
- ガンは 6 死因中 県差最小 (CV 6.37% 男年齢調整、1 位)
- 糖尿病・肺炎・脳血管疾患の方が CV で 2-3 倍大きい

**この「逆認知」は base rate effect + UI color scale + 注目バイアス + ガン部位別ナラティブの引きずり** で説明可能。

**MedIntel への示唆**: 「県差の大きさ」を伝える UI が現状不十分な可能性。CV / max-min 比などの **県差 KPI** の併記を検討する余地がある。
