## 0. 結論

> 中四国・九州10県の **すべて (10/10) が支援型と判定** された。仮説「中四国・九州の一部県では在宅移行 proxy が相対的に厚い」は **完全に支持** される。  
>
> しかし重大な発見として、**「供給 proxy が厚くても、結果指標 (年齢調整死亡率) は改善されていない」** ことが定量的に明らかになった。  
> 山口県/徳島県/鹿児島県では、肺炎・心疾患・腎不全の年齢調整死亡率が全国平均を大きく上回る。  
>
> → **「支援型 = 良い医療」という単純解釈は誤り**。「供給」と「結果」は別軸であり、Bridge 解釈仕様の peer review 原則と整合する。

## 4. 重大な発見 (Devil's Advocate)

### 4.1 「供給+」と「結果」は別軸
| 県 | 供給判定 | 死亡率の傾向 |
|---|---|---|
| 山口県 | ✅ 強い支援型 (4/5) | **肺炎+31%、心疾患+14%、腎不全+17% で結果は悪い** |
| 徳島県 | ✅ 強い支援型 (4/5) | 肺炎+32%、腎不全+19% で結果は悪い |
| 鹿児島県 | ✅ 強い支援型 (4/5) | 肺炎+26% で結果は悪い |
| 熊本県 | ✅ 支援型 (3/5) | 脳血管-9%、悪性-9% で結果は良好 |
| 岡山県 | ✅ 支援型 (3/5) | 全死因が全国平均近くで安定 |
| 島根県 | ✅ 支援型 (3/5) | **肺炎-35% で結果は良好** |

→ **「支援型 = 結果も良い」という単純な構図は成り立たない**。

### 4.2 肺炎死亡率の高さ
- 山口/徳島/鹿児島で肺炎年齢調整+25%以上
- これらの県は高齢化が高く、誤嚥性肺炎の集計差や呼吸器管理の地域差等が交絡している可能性

### 4.3 慢性期病床の「過剰」?
- 高知 +136%、熊本 +65%、徳島 +54%、鹿児島 +47%
- 病床数は厚いが、必要量に対して過剰の可能性
- **「在宅移行が進んでいない」と読むこともできる** (慢性期入院依存)

|---|---|---|

**観察**: 高齢化レベルはほぼ同等 (両群とも 18-22%) だが、供給 proxy の地域差は極めて顕著。

### 5.1 結果指標 (年齢調整死亡率) の対照

| 死因 | E-2 東北 (傾向) | E-3 中四国・九州 (傾向) |
|---|---|---|
| 脳血管疾患 | **+20-54% (全県悪化)** | -9% 〜 +7% (混在、改善傾向) |
| 腎不全 | -1% 〜 +35% (悪化傾向) | -8% 〜 +27% (混在) |
| 肺炎 (E-3で集計) | (集計なし) | **-35% 〜 +32% (極度の混在)** |

→ **脳血管疾患では E-2 vs E-3 で顕著な差**。供給 proxy と結果指標が並列。  
→ **腎不全・肺炎では群内 ばらつきが大きい**。供給だけでは説明できない他の要因が存在。

## 6. 慎重な留保 (peer review遵守)

- ❌ 「支援型 = 医療の質が高い」と断定しない
- ❌ 「在宅医療 cap が厚い → 結果が良い」という単純因果推論をしない
- ✅ 「供給 proxy が厚くても、結果指標は改善されない場合がある」という事実を記録
- ✅ proxy 表現 (cap.homecare = 在宅医療施設基準届出件数の集計値) を維持
- ⚠️ 慢性期病床の +50% 以上は「過剰」「在宅移行の遅れ」とも解釈可能 (両義的)
- ⚠️ 死亡率の地域差には文化・行政・医療提供体制の交絡が含まれる
- ⚠️ E-2 ギャップ型 vs E-3 支援型の対照は **「供給」軸では明確** だが、**「結果」軸では完全な対照にはならない**

## 7. MedIntel への含意

### 7.1 Bridge Risk Model v1 への影響
- 「供給 proxy が厚い」と「結果指標が良い」は **独立に評価すべき**
- E-3 結果は、Bridge の各列を **独立軸として読む** ことの正当性を補強

### 7.2 仮説生成装置としての価値
- E-2 + E-3 で「**供給 proxy** と **結果指標** の不一致パターン」を **2地域群で実証**
- 沖縄 (E-1) も同種の不一致パターン
- → MedIntel は単一指標で地域を評価せず、**多軸で見る仮説生成・構造探索ツール**として機能

### 7.3 表現の慎重化
- 「支援型県」を「在宅 proxy が厚い県」と記述する
- 「ギャップ型県」を「在宅 proxy が薄く、かつ脳血管調整死亡率が高い県」と記述する
- 「医療の質が高い/低い」という単純評価を避ける
