## 0. 本書の性質

本書は Bridge v0 の解釈ガイドであり、5県(東京/大阪/高知/沖縄/北海道)の体系的比較から導出された**仮説候補**を記録する。

**重要**: Bridge v0 は **仮説生成装置** (hypothesis generator) であり、**異常検出システム** ではない。
- 単純な「整合」「不整合」の断定をしない
- 「リスク高×受療低」「受療低×処方高」等のパターンは**仮説候補**として扱い、追加データでの検証を要する
- 粗死亡率(年齢調整前)は医療アウトカムの優劣として直接解釈しない

## 2. 主要仮説 (4件)

### 仮説1: 沖縄県の糖尿病パラドックス

**観察**:
- 体重増加リスク **+23% (全47県中最高)**
- 受療率 **-47%**
- 処方proxy **-72%**
- 死亡率 -11%

**矛盾**: 「リスクが顕著に高い」のに「受療と処方が顕著に低く」、「死亡率も低め」。

**候補仮説** (peer review指摘の5要因):
1. **受療構造の違い**: 有病率は高いが医療機関受療率が低い (生活習慣による未受診)
2. **薬効分類漏れ**: インスリン・GLP-1・SGLT2等が `396` 外に分類されている可能性
3. **年齢構成**: 沖縄の高齢化率は全国より低く、糖尿病死亡率(年齢調整前)に表れにくい
4. **医療アクセス**: 離島・地理的アクセス問題
5. **データ仕様**: NDB集計の取り込み範囲差 (内服/注射/院内外)

**Phase 2 検証手順**:
- 396 + 249 (その他のホルモン剤) で再集計し、インスリン分類漏れを定量化
- 都道府県別 糖尿病有病率データ (NDB特定健診結果, HbA1c高値率) と接続
- 年齢調整死亡率データの入手可能性確認
- 沖縄の県内地域差 (本島/離島) を二次医療圏粒度で確認

### 仮説2: 北海道の喫煙率と結果

**観察**: 喫煙率 **+27% (突出)** → 循環器処方 +16%, がん死亡率 **+28%**

**仮説**: 喫煙率の高さが循環器・がん両方の結果指標に反映している可能性。

**注意**: 寒冷気候・食生活・経済状況など他要因も同時に作用しうる。喫煙率単独の寄与は分離不可。

### 仮説3: 高知県の高齢化パターン

**観察**: 高知の特徴的な値
- 循環器処方 **+51% (突出)**
- 糖尿病処方 **+87% (突出)**
- 循環器死亡率 **+35%**
- がん死亡率 **+20%**

**仮説**: 高齢化県の典型パターン。年齢調整なしの粗死亡率と高齢者向け処方量が同方向に振れる。

**注意**: 死亡率は粗死亡率なので、医療の優劣ではなく**人口構造の差**として読むべき。

### 仮説4: 東京の大都市・高機能・若年型

**観察**: 全領域でリスク低/受療率低/高機能急性期供給高/死亡率低

**仮説**: 大都市・若年・医療アクセス良の複合効果。

**注意**:
- 死亡率の低さは年齢構成の影響が支配的 (大都市は若年人口比率高)
- 「東京の医療が優秀だから死亡率が低い」とは結論できない
- 高機能急性期病院の集積は事実だが、それが住民全体の死亡率改善に直結するかは別問題

## 3. v0 構造的限界 (peer review指摘事項)

### 3.1 数量合算の限界

NDB処方薬データは「薬効分類別数量」であり、薬剤ごとに単位・剤形・用量が異なる。

**やってはいけない解釈**:
- ❌ 「処方proxy +51% = 治療量1.5倍」
- ❌ 「処方proxy = 患者数」
- ❌ 「異なる薬効分類の合算 = 治療強度」

**正しい解釈**:
- ✅ 「47都道府県の中で処方量が相対的に高い/低い」(順位的傾向)
- ✅ 「地域間の処方パターンの差」
- ✅ 仮説生成のトリガー

### 3.2 比較基準の人口非加重性

- ndbQ・ndbRx に '全国' エントリがないため 47都道府県の単純平均を代理使用
- 真の全国値 (人口加重) とは数pt差の可能性
- 患者調査・bedFunc・vitalStats は national 直接値を使用 (人口加重済)

### 3.3 結果指標の限界

- vitalStats 死因別死亡率は **粗死亡率(年齢調整前)**
- 年齢構成の影響を強く受ける
- 「医療アウトカムの優劣」として直接解釈しない

### 3.4 供給proxy の非疾患専用性

- 「高度急性期+急性期」「高度急性期」「がん関連capability」はいずれも疾患専用ではない
- 例: cap.surgery は循環器/整形/外科を含む
- proxyラベルで明示済

### 3.5 「医療利用」=処方薬proxyの限界

- 処方薬proxyは外来処方が中心 (院内処方・注射・手術等は別データ源)
- 入院治療・処置・手術・放射線は **未含**
- がん治療では分子標的薬・免疫CPI が薬効分類429に含まれるかは要追加検証

## 4. domainMapping v0 FROZEN 宣言

`lib/domainMapping.js` の 3領域定義を **v0 FROZEN at commit 475a174** とする。

### FROZEN の意味 (重要 — peer review feedback反映)

> FROZEN は「**医学的に正しい構成の確定**」ではなく、「**現時点の解釈仕様の固定**」を意味する。
>
> 新データ・年齢調整死亡率データ・薬剤名ベース補正・NDB追加カテゴリ投入時には、
> FROZEN解除 commit + interpretation notes 更新で再評価する。

つまり、3領域の codes/keys は「現時点の v0 として動作上固定」されているが、
医学的・統計的により良い構成が判明すれば積極的に解除して更新すべきもの。

### FROZEN 規則

1. 3領域 (cardiovascular / diabetes_metabolic / cancer) の `risk` / `demand` / `utilization` / `supply` / `outcome` のキー構造は変更不可
2. codes / patientSurveyKey / bedFuncKeys / vitalCause の追加・削除は不可
3. 変更したい場合: 別の commit で **FROZEN解除宣言** + 変更内容 + interpretation notes 更新を1セットで実施

### 変更が許可される範囲

- ❌ 3領域の指標定義 (codes/keys)
- ❌ 比較基準のロジック (47県平均/全国値)
- ✅ note / proxyLabel / label の文言修正
- ✅ describeDelta() の閾値調整 (デフォルト ±5%/±15%)
- ✅ 新規4領域目以降の追加 (脳血管/呼吸器/腎疾患)

## 5. 脳血管領域 v1 (4領域目, 2026-04-28追加)

### 5.1 設計判断

レビュー指定通り、抗血栓薬は循環器との重複が大きいため脳血管専用治療proxyに使わない。
データ実在確認の結果、患者調査では脳血管が独立カテゴリとして公開されておらず、
受療率セルは demand=null + 注記表示で対応。

| 列 | 採用 | 不採用 |
|---|---|---|
| リスク | 喫煙率 (smoking) | 血圧データ (NDB公開未収載) / 運動不足はv2 |
| 疾病負荷 | **null + Note** (患者調査独立データなし) | Ⅸ_循環器系の流用 (循環器と完全重複で誤読リスク) |
| 医療利用 | **null + Note** | 抗血栓薬(333) — 循環器との重複大 |
| 供給proxy | 高度急性期+急性期+回復期 床シェア | 「脳卒中対応施設数」断定 |
| 結果 | 脳血管疾患死亡率 (粗死亡率) | 医療品質と直接結びつけ |

### 5.2 5県 sanity (脳血管 v1)

| 県 | 喫煙率 | 供給(高+急+回) | 結果(脳血管死亡率) |
|---|---|---|---|
| 全国/47県平均 | 22.0% | 72.4% | 85.5 |
| 東京 | 19.9% (-10%) | 77.7% (+7%) | 62.3 (-27%) |
| 大阪 | 22.0% (-0%) | 70.5% (-3%) | 66.5 (-22%) |
| 高知 | 22.7% (+3%) | 57.5% (-20%) | **116.8 (+37%)** |
| 沖縄 | 19.6% (-11%) | 72.4% (+0%) | 70.2 (-18%) |
| 北海道 | **28.0% (+27%)** | 63.9% (-12%) | 99.0 (+16%) |

### 5.3 観察される仮説

**仮説5: 高知県の脳血管死亡率高 + 供給薄パターン**
- 結果+37% (47県中相当上位) かつ 供給-20%
- 高齢化県の典型: 急性期+回復期供給が薄く、結果指標が高い
- 注意: 粗死亡率なので年齢構成の影響大

**仮説6: 北海道の喫煙×供給薄 + 結果高**
- 喫煙+27% (突出) と 結果+16% / 供給-12% の組み合わせ
- 喫煙は脳血管疾患の主要risk factorとされるが、本データから因果は分離不可
- 寒冷気候・食生活との交絡を検討すべき

**仮説7: 大都市 (東京・大阪・沖縄) の死亡率低**
- 3都市すべて結果-18%以下
- 沖縄含む大都市・若年構成の整合性
- 高度急性期+急性期+回復期供給は大都市が手厚い (東京+7%等)
- ただし、大阪も結果-22%だが、大阪は供給-3%でほぼ全国並 → 死亡率低下の主因は若年人口構成と推測

### 5.4 v1 構造的限界

- 患者調査の脳血管独立データが NDB公開データに未収載 → 疾病負荷proxy空白
- 抗血栓薬を採用しないため、医療利用列も空白 → 5列中2列が空白で情報量弱い
- 供給は機能区分床ベースで cap.acute/cap.rehab を含まない (Bridge UI現実装の制約)
- 脳血管疾患の結果は心疾患の半分以下の死亡率 → 数値スケールの違いを考慮要

## 6. 呼吸器領域 v1 (5領域目, 2026-04-28追加)

### 6.1 設計判断

レビュー指定「結果は肺炎/誤嚥性肺炎/慢性下気道疾患のいずれか」に対し、
データ実在確認の結果:
- ✅ 肺炎: 66.6 (vital_stats)
- ✅ 誤嚥性肺炎: 52.9 (vital_stats)
- ❌ 慢性下気道疾患: 公開vital_statisticsに未収載

→ 肺炎 + 誤嚥性肺炎 を「高齢者呼吸器アウトカムproxy」として2行縦並びで表示。

| 列 | 採用 | 不採用 |
|---|---|---|
| リスク | 喫煙率 (smoking) | 「喫煙率高=COPD多い」と断定 |
| 疾病負荷 | Ⅹ_呼吸器系の疾患 受療率(外来) | — |
| 医療利用 | **225 気管支拡張剤のみ** | 222鎮咳/223去たん/224鎮咳去たん (風邪等軽症処方支配的) |

| 結果 | 肺炎死亡率 + 誤嚥性肺炎死亡率 (2行縦並び) | 慢性下気道疾患 (データ無) / 3指標雑な合算 |

### 6.3 観察される仮説

**仮説8: 北海道の処方+100% × 受療率-39% の不整合**
- 受療率は全国最低レベル(-39%)なのに気管支拡張剤処方は全国2倍
- 候補仮説:
  1. 地理的アクセス困難で1回受療あたりの集約処方
  2. 喫煙率+27%で COPD等慢性疾患患者が多く、長期処方
  3. 受療率の集計方法と処方データ集計の地域差
- v0 同様、「異常」ではなく「仮説候補」として扱う

**仮説9: 高知県の高齢化 × 呼吸器**
- 処方+68% / 肺炎死亡+65% / 誤嚥+23%
- 既存仮説3(高知の高齢化)と整合
- 高齢者誤嚥性肺炎は摂食機能低下と相関 → 高齢化県で増加するパターン

**仮説10: 東京の受療率+28% × 死亡率-29%**
- 受療率は全国比+28%で高いが肺炎死亡率は-29%
- 大都市は外来受療しやすく(受療率高)、若年人口主体(死亡率低)
- 仮説4(東京の若年人口主因)と整合

**仮説11: 大阪の処方-22% × 死亡率+19%/+28%**
- 処方は低いのに死亡率は両方とも高い
- 候補仮説:
  1. 受療しても処方されにくい (適切な治療提供差)
  2. 高齢化率が東京より高く、結果に表れる
  3. データ集計の偶然
- 単独で結論不可、追加検証要

### 6.4 v1 構造的限界

- **慢性下気道疾患 (COPD等) 死亡率が公開データに未収載**: 呼吸器領域の最重要結果指標が欠落
- **供給proxy 空白**: 5列中の供給列がnull (在宅酸素・呼吸器内科供給は v2 検討課題)
- **薬剤proxy が225単独**: 鎮咳/去たん系は除外したため、軽症呼吸器疾患のシグナルは捕えない
- **肺炎+誤嚥性肺炎の合算解釈**: 縦並び表示で個別値を見せるが、ユーザーの読み方として「合算してざっくり判断」される懸念
